## File Systems in the Cloud (AWS SimuLearn)

I worked with **shared file systems** to manage data that requires concurrent access across multiple cloud instances. This included configuring file system mounts, managing permissions, and understanding how these systems provide persistent, high-performance storage for applications with shared data needs.

The labs emphasized practical management techniques such as monitoring file system performance, configuring backups, and implementing security controls to protect sensitive data. Additionally, I developed an understanding of **storage evaluation**, considering factors such as access patterns, durability, availability, performance, and cost. This enabled me to select appropriate storage solutions based on application requirements and business objectives.
