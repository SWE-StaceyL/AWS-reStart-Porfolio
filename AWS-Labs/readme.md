# AWS re/Start Canvas Labs 

## Overview
This repository contains hands-on lab work completed as part of the **AWS re/Start Programme** — a project-based learning path focused on cloud fundamentals, Linux, networking, security, and Python scripting. Labs are organised by domain and reflect real-world AWS use cases aligned with the **AWS Certified Cloud Practitioner (CLF-C02)** exam objectives.


## Repository Structure

| Folder | Description |
|---|---|
| `Compute` | EC2 instance provisioning and configuration |
| `Databases` | Amazon Aurora setup and management |
| `Linux` | Linux fundamentals and file system operations |
| `Networking` | VPC, Apache HTTP server, and EC2 network configuration |
| `Python` | Python scripting challenges on AWS |
| `Scaling and Name Resolution` | Auto Scaling and Route 53 routing policies |
| `Security` | Firewall configuration and malware concepts |
| `Servers` | Server setup and management |

## Completed Labs

### Compute — EC2 Instance
**Folder:** `Compute/`
- Launched and configured an Amazon EC2 instance
- Documented instance setup steps and best practices
- Explored instance types, key pairs, and security group configuration

---

### Linux — File Systems & Fundamentals
**Folder:** `Linux/`
- Navigated the Linux CLI using core commands (`ls`, `cd`, `mkdir`, `rm`, `grep`)
- Managed file permissions with `chmod` and `ls -la`
- Performed user management using `useradd`, `usermod`, `userdel`, and `chown`
- Used built-in help tools: `man`, `whatis`, and `apropos`
- Worked with the `nano` text editor and file system operations

---

### Networking — Apache HTTP Server on EC2
**Folder:** `Networking/`
- Configured an Apache HTTP server on an EC2 instance
- Applied a troubleshooting guide covering common connectivity and configuration issues
- Worked with security groups to manage inbound and outbound traffic rules

---

### Python — Scripting Challenge
**Folder:** `Python/`
- Wrote a Python script to compute prime numbers using a square root optimisation approach
- Executed the script in the AWS Cloud9 environment on EC2
- Practised Python fundamentals: loops, conditionals, and algorithmic optimisation

---

### Scaling and Name Resolution — Route 53 Failover Routing
**Folder:** `Scaling and Name Resolution/`
- Configured **Route 53 Failover Routing** with primary and secondary records
- Set up **health checks** to monitor endpoint availability
- Simulated failover scenarios to validate DNS-based traffic redirection
- Explored routing policies: Simple, Failover, Weighted, and Latency-based

---

### Security — Firewall & Malware
**Folder:** `Security/`
- Studied firewall concepts and rule-based traffic filtering
- Reviewed malware types and cloud security best practices
- Aligned concepts with AWS security services (Security Groups, NACLs)

---

### Databases — Amazon Aurora
**Folder:** `Databases/`
- Set up and configured an Amazon Aurora database instance
- Documented lab steps with supporting images
- Explored Aurora's high availability and read replica features

---
